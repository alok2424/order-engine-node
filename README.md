# order-engine-node
# 

##  Overview
This project implements a **fault-tolerant order processing system** that ensures:
- Data consistency
- Concurrency safety
- Idempotency
- Reliable rollback on failure

The system simulates a real-world e-commerce backend where multiple users can place orders concurrently.

---

##  Tech Stack

- Node.js
- Express.js
- Sequelize ORM
- SQLite (can be replaced with PostgreSQL/MySQL)
- JavaScript (ES6)
